# Paper (KDD / ACM `acmart`)

- `main.tex` — the paper in ACM `acmart` `sigconf` format (KDD's required style).
- `refs.bib` — bibliography.
- `figures/` — figures, regenerated by `python src/make_figures.py` from `results/*.json`.

## Compile (Overleaf — easiest)
1. New Project → Upload Project → zip this `paper/` folder (with `figures/`).
2. Menu → Compiler: **pdfLaTeX**; TeX Live **2021 or newer**.
3. Recompile. `acmart` is preinstalled on Overleaf.

## Compile (local)
```bash
cd paper
pdflatex main && bibtex main && pdflatex main && pdflatex main
```
(Requires a TeX distribution with the `acmart` class, e.g. TeX Live full.)

## Submitting to KDD
- Current class options are `[sigconf, review, nonacm]`. For the **double-blind** submission KDD
  requires, use `[sigconf, review, anonymous]` and remove author-identifying text.
- For the **camera-ready**, drop `review`/`anonymous`, fill in real `\author`/`\affiliation`/`\email`,
  add the ACM copyright/DOI block from the ACM rights email, and select CCS concepts.
- Check the current KDD call for the page limit and Research vs.\ Applied Data Science track.
